# Battleship

## Overview

Battleship game project with game logic modules, unit tests, and a browser UI.

## Tech Stack

- JavaScript (ES Modules)
- HTML
- CSS
- Jest
- Babel

## Project Files

- `index.html`
- `script.js`
- `src/`
- `__tests__/`

## How to Run

```bash
npm install
npm test
```

Open `index.html` in your browser for the UI.

## Status

In progress and playable as an MVP.
